# Contributing to NileGazer

Thanks for helping improve NileGazer.

## Ways to contribute

- Fix bugs.
- Improve docs.
- Add tests.
- Build adapters.
- Improve examples.
- Add safety and policy checks.
- Suggest roadmap items.

## Before you start

- Read the README.
- Check open issues.
- Look for `good first issue` and `help wanted`.
- Keep pull requests small and focused.

## Development flow

1. Create a branch.
2. Make your changes.
3. Test locally.
4. Open a pull request.

## Communication

Be respectful and constructive in all discussions.
